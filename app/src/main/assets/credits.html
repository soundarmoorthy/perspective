On-line Resources

One advantage of Android is the ready availability of numerous examples, both online and in print. This demo used code from a number of sources as starting material for many of it's functions. Online sources include:

  • developer.android.com. Note that Android source code and documentation are licensed under terms of the Apache License. We used both in creation of this application.
  • The GraphicScreenDump Java class is based upon code found at: http://www.anddev.org/how_to_get_opengl_screenshot__useful_programing_hint-t829.html (and other locations) by "eugenk".
  • The preferences code was based on an example found at: http://www.javacodegeeks.com/2011/01/android-quick-preferences-tutorial.html
  • The graphics renderer class was based on code found at http://www3.ntu.edu.sg/home/ehchua/programming/android/Android_3D.html.
  • Typhoon Labs Open GL ES Tutorials at ftp://raphaello.univ-fcomte.fr/pub/OpenGL-ES

Books You May Find Useful

Some excellent books on the topic include:

  • The Android Developer's Cookbook: Building Applications with the Android SDK, by James Steel and Nelson To, published by Addison-Wesley, 2011
  • Android Recipes, A Problem Solving Approach, by Dave Smith and Jeff Friesen, published by Apress, 2011
  • Professional Android Sensor Programming, by Greg Milette and Adam Stroud, published by John Wiley & Sons, Inc, 2012
  • Advanced Graphics Programming Using OpenGL, by Tom McReynolds and David Blythe, published by Morgan Kaufmann Publishers, 2005
  • Programming Android 2nd edition, by Zigurd Mednieks, Laird Dornin, G. Blake Meike & Masumi Nakamura, Published by O'Reilly, 2012
  • Beginning OpenGL Game Programming, by Dave Astle and Kevin Hawkins, published by Premier Press, 2004
  • Android Essentials, by Chris Hasemann, published by Apress
  • Pro OpenGL ES for Android, by Mike Smithwick and Mayank Verma, published by Apress
  • Android Application Development for Dummies, by Donn Felker and Joshua Dobbs, published by Wiley Publishing, Inc., 2011
  • Professional Android Application Development, by Reto Meier, pubished by Wiley Publishing, Inc., 2009
  • The Busy Coder's Guide to Advanced Android Development, by Mark L. Murphy, published by CommonsWare books, 2009
